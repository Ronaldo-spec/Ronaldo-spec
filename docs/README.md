# Portfolio Page

This folder contains the static portfolio page for Ronaldo Firmansyah.

## Files

- `index.html` - main portfolio page
- `styles.css` - visual styling
- `script.js` - subtle scroll interaction

## Recommended GitHub Pages Setup

To publish this page as a GitHub Pages portfolio:

1. Open repository settings for `Ronaldo-spec/Ronaldo-spec`.
2. Go to **Pages**.
3. Set source to **Deploy from a branch**.
4. Select branch: `main`.
5. Select folder: `/docs`.
6. Save.

After GitHub Pages finishes deployment, the portfolio should be available at a URL similar to:

```text
https://ronaldo-spec.github.io/Ronaldo-spec/
```

If a dedicated user site repository named `Ronaldo-spec.github.io` is created later, these files can be moved there so the portfolio URL becomes:

```text
https://ronaldo-spec.github.io/
```
